Draggable timeline.

Objects

OAT.Timeline(contentDiv, options)


Creates a Timeline inside contentDiv . The options may include:

  • lineHeight - int, height of one record
  • bandHeight - int, height of one band's header
  • margins - int, (default 200 )
  • resize - bool (default true )
  • formatter - bool (default true )
  • sliderHeight - int, height of slider
  • autoHeight - bool, should height be automatically adjusted? (default true )

Methods

OAT.Timeline.addBand(name, color, label)


Adds a data band to chart. The band is identified by name , drawn in color , and labeled with label .
